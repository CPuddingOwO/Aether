## What is Aether?

Aether is a 2D game engine written in C++20 with CMake. It is based on the following concepts:

* Using SDL3 and Vulkan
* Using ImGui for UI
* Using FLECS for ECS
* Using C++20
* Using CMake
* Using Conan2
